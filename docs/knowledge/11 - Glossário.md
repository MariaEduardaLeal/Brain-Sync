# Glossário

Termos específicos do domínio do Brain-Sync Desktop.

| Termo | Definição |
| :--- | :--- |
| **Antigravity** | O agente de IA cujos artefatos estão sendo monitorados e organizados. |
| **Brain Folder** | Pasta do sistema localizada em `.gemini/antigravity/brain` onde ficam os logs de raciocinio. |
| **Vault (Cofre)** | Diretório de destino (`BrainSyncVault`) onde os dados são arquivados permanentemente. |
| **Session (Sessão)** | Um ciclo completo de interação com a IA (Task -> Plano -> Execução). |
| **Artifact (Artefato)** | Documentos gerados durante a sessão (ex: `task.md`, `walkthrough.md`). |
| **Resolved (.resolved)** | Versões intermediárias de um artefato geradas pela IA antes da finalização. |
| **Scan (Varredura)** | O processo de ler a pasta do Antigravity e importar novos dados para o projeto. |
| **Match Heuristic** | Lógica de associação automática entre uma sessão e um projeto baseada nos caminhos de arquivos. |
