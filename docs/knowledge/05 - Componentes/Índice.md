# Componentes (Frontend)

O frontend é construído com **Vue 3** e **Bootstrap 5**, focado na visualização de dados e gerenciamento de projetos.

## 🧱 Lista de Componentes

- [[05 - Componentes/App|App.vue]]: Componente raiz que gerencia o estado global da UI e IPC.
- [[05 - Componentes/AIFlowchart|AIFlowchart.vue]]: Renderiza uma sessão de raciocínio completa com histórico.
- [[05 - Componentes/SessionGraphModal|SessionGraphModal.vue]]: Modal para visualização detalhada de grafos de sessões.

## 🎨 Design System

- **Tema**: Dark-mode com gradientes neomórficos e glassmorphism.
- **Tipografia**: 'Segoe UI', sans-serif.
- **Cores Principais**:
    - Fundo: `#060814` a `#05070f`.
    - Destaque: `#8bd4ff` (Azul claro), `#7c3aed` (Violeta).
    - Sucesso: `#10b981`.
    - Perigo: `#ef4444`.
